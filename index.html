KeyBoard Hero

About

This game was created as a semestral project. Its purpose was to learn and play arround with modern web technologies such as HTML5, CSS and JavaScript.

The game

The main goal of the game is to get as high score as possible. Using keyboard, pressing the right letter at the right time will give you points.

Earning points

  • +10 successful press
  • -5 miss press
  • -1 the box goes under the line

More points can be earn by successful presses in a row. The bonus points are represented by a multiplication constant ranges from 1 to 3. And be careful the speed is increasing with higher points.
